Notes:
The file is condensed to one row per unique accession outcome.
download_request_accessions records the terminal exact accession token or tokens
used for the grouped row.
final_accession records the realised versioned accession from the extracted
payload when one exists.
The workflow treats the realised accession as authoritative, so versioned request accessions fail closed if the exact realised accession is absent after extraction.
Only versionless request accessions may accept a unique same-family realised
version.
